# Periodic Table App

A mobile-first learning tool for the periodic table with two modes: flashcards and multiple choice quiz. Installable as a Progressive Web App.

Built as a portfolio project to demonstrate UX process alongside front-end development.

![CI](https://github.com/KerryCX/periodic-table/actions/workflows/ci.yml/badge.svg)

## Status

Live at [periodic-table.kerryclements.com](https://periodic-table.kerryclements.com)

## Tech stack

- React
- TypeScript
- Vite
- CSS
- Vitest
- Figma
- WCAG 2.2 AA
- PWA (Vite PWA plugin)

## Run locally

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```

Coverage report:

```bash
npm run coverage
```

## About

This project was rebuilt from an earlier flashcard app after studying UX.
The design was completed in Figma with WCAG 2.2 AA accessibility checking before any code was written.
The result is a mobile-first PWA used for real exam revision, installable on Android, iOS, and desktop via Chrome.
